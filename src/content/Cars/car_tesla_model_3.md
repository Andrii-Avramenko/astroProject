# 2023 Tesla Model 3

## Summary
The Tesla Model 3 is an all-electric fastback sedan that revolutionized the EV market. The 2023 version features sleek minimalist styling, exceptional range, and rapid acceleration.

## Advantages
- Outstanding electric range and industry-leading Supercharger access
- Impressive, immediate acceleration and sporty handling

## Disadvantages
- Minimalist interior relies entirely on the touchscreen, which can distract drivers
- Build quality and panel gaps can be inconsistent
